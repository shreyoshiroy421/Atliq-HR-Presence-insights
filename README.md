# Atliq-HR-Presence-insights
A power BI dashboard analyzing employee attendance and WFH trends

# Dashboard Preview
![Dashboard Screenshot](path/to/your/screenshot.png) 

# Project Overview
This project provides a comprehensive analysis of employee attendance patterns at Atliq Technologies. 

# Key Metrics (from the dashboard)
- **Attendance Rate:** 91.7%
- **Work From Home (WFH) %:** 10.0%
- **Sick Leave (SL) %:** 1.1%

# Tools Used
- **Power BI Desktop:** For report generation and visualization.
- **Power Query:** For data cleaning and transformation.
- **DAX:** For calculating complex measures like % of Attendance over time.
